# 🤖 Gemini AI Chatbot

An AI-powered chatbot built using **Google Gemini API** and **Streamlit**. This chatbot can answer user queries in real-time using Google's latest Gemini models.

## 🚀 Features

* AI-powered conversational chatbot
* Real-time responses using Google Gemini API
* Simple and interactive Streamlit UI
* Secure API key management using environment variables
* Easy to set up and run locally

## 🛠️ Tech Stack

* Python
* Streamlit
* Google Gemini API
* Python Dotenv

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/gemini-ai-chatbot.git
cd gemini-ai-chatbot
```

### 2. Create Virtual Environment

```bash
python -m venv venv
```

Activate the environment:

**Windows**

```bash
venv\Scripts\activate
```

**Mac/Linux**

```bash
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Create Environment File

Create a `.env` file in the project root directory:

```env
GOOGLE_API_KEY=YOUR_GEMINI_API_KEY
```

### 5. Run the Application

```bash
streamlit run app.py
```
